# Frontend - Agrosend

## Descripción

Frontend completo del sistema de gestión de inventario agrícola **Agrosend**, desarrollado con HTML5, CSS3 y JavaScript vanilla (ES6+).

## Estructura de Carpetas

```
frontend/
├── index.html
├── login.html
├── css/
│   ├── styles.css
│   └── dashboard.css
├── js/
│   ├── api.js
│   ├── auth.js
│   ├── app.js
│   ├── dashboard.js
│   ├── productos.js
│   ├── ventas.js
│   ├── mercado.js
│   └── clientes.js
└── img/
```

## Características Principales

✅ **Autenticación** - Login seguro con persistencia de sesión
✅ **Dashboard** - Gráficos interactivos con Canvas HTML5
✅ **CRUD Completo** - Productos, Ventas, Clientes, Mercado
✅ **Búsqueda en Tiempo Real** - Sin recargas de página
✅ **Modales Interactivos** - Crear/editar registros
✅ **Tema Oscuro/Claro** - Preferencias del usuario
✅ **Responsive Design** - Mobile-first approach
✅ **Validación de Datos** - Cliente y servidor
✅ **Manejo de Errores** - Mensajes amigables al usuario

## Instalación

1. Coloca los archivos en tu servidor web
2. Asegúrate de que la API está disponible
3. Abre `index.html` o `login.html` en el navegador

## Archivos Principales

- `index.html` - Dashboard principal
- `login.html` - Página de login
- `css/styles.css` - Estilos globales
- `css/dashboard.css` - Estilos específicos
- `js/api.js` - Llamadas a API
- `js/auth.js` - Autenticación
- `js/app.js` - Navegación principal
- `js/dashboard.js` - Dashboard
- `js/productos.js` - Gestión de productos
- `js/ventas.js` - Gestión de ventas
- `js/mercado.js` - Comparación de precios
- `js/clientes.js` - Gestión de clientes

## Características Técnicas

- JavaScript ES6+
- Sin dependencias externas
- Fetch API para HTTP
- LocalStorage para sesiones
- Canvas HTML5 para gráficos
- CSS3 Grid y Flexbox
- Animaciones suaves
- Performance optimizado
